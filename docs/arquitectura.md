## Explicación técnica
La aplicación funciona mediante una arquitectura *cliente-servidor*:

- El **frontend** se ejecuta en el navegador
- El **backend** procesa pedidos
- La **base de datos** guarda usuarios y productos

## Tecnologías

| Tecnología | Uso |
|----------|-----|
| React | Interfaz |
| Node.js | Servidor |
| MySQL | Base de datos |
| Stripe | Pagos |
