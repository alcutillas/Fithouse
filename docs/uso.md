# Uso

## Usuarios tipo

| Usuario        | Función |
|---------------|--------|
| Cliente       | Compra suplementos |
| Administrador | Gestiona productos |
| Visualizador    | Visualiza la pagina |

## Casos de uso

1. El cliente se registra
2. Elige un producto
3. Lo añade al carrito
4. Hace el pedido
