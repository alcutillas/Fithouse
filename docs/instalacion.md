## Requisitos
Para ejecutar la tienda necesitas:

- Node.js
- Base de datos MySQL
- Navegador web
- Git

## Pasos de instalación

1. Clonar el repositorio  
2. Instalar dependencias  
3. Configurar el entorno  
4. Ejecutar el servidor  

```bash
git clone https://github.com/tienda-suplementos/app.git
cd app
npm install
npm run start
